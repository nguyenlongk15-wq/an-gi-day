# Ăn Gì Đây?

App mini game chọn món ăn, viết bằng Expo + React Native + TypeScript.

## Chạy bản web

```bash
npm run web
```

## Chạy Android/iOS bằng Expo Go

```bash
npm start
```

Sau đó quét QR bằng Expo Go. iOS cần Expo Go trên iPhone/iPad; Android cần Expo Go trên Android.

## Cấu trúc chính

- `src/data`: câu hỏi và món ăn local
- `src/utils`: random, quiz engine, scoring, storage
- `src/screens`: Home, Quiz, Result, Favorites
- `src/components`: các khối UI dùng lại
- `assets/food-hero.png`: ảnh hero của app
